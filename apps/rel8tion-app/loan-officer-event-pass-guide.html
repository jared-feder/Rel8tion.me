REL8TION Event Pass Field Guide

Scan. Tap. Activate.

The Event Pass is already made. The loan officer or agent uses the printed QR and the NFC chip on that same physical Event Pass to attach it to one open house.

QR first → NFC on the same Event Pass → confirm the open house and agent → activate
1

Scan the printed QR

Open the phone camera and scan the QR printed on the Event Pass keychain.

2

Tap the same Event Pass

Tap the NFC chip on that same physical keychain. Do not use a different keychain.

3

Choose open-house mode

Select At a Listing or Open House. Keep setup on the same phone and browser.

4

Confirm the listing

Allow location access, search by address or MLS, or enter the listing manually.

5

Confirm the hosting agent

Verify the correct agent and profile before activating.

6

Activate the listing

Confirm the open house and tap Activate This Listing. The Event Pass is now live.

7

Use the event dashboard

Future NFC taps on the activated Event Pass open the agent dashboard for this event.

8

Buyers scan the QR only

Buyers do not tap NFC. They scan the QR and complete check-in choices and required disclosure acknowledgements.

9

Agent receives the alert

After the buyer completes check-in and required disclosures, the hosting agent receives the buyer information by text.

10

Financing stays opt-in

Only when the buyer requests financing help, the assigned live loan officer receives an alert and the buyer receives the loan officer introduction.

Important: An Event Pass is for one open-house event. It is not a public agent profile. Buyer financing information is shared with the loan officer only when the buyer requests financing help.
