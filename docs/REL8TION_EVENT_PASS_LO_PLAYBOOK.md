# REL8TION Event Pass Loan Officer Playbook

Last updated: 2026-05-16

## Purpose

This playbook explains how loan officers should use the Rel8tion Event Pass at open houses, how to explain it to agents, and how to convert the live event into future agent relationships and Rel8tion kit/dashboard sales.

The Event Pass is a B2B open-house technology pass. Loan officers purchase verified profile placement, availability placement, and Event Pass access so they can support live open-house events when buyers request financing help.

The buyer lead is not the product. The product is B2B open-house technology, live event support, LO availability placement, communication routing, event reporting, and post-event conversion.

## Physical Event Pass Usage

The Event Pass has two sides:

| Side | Purpose |
| --- | --- |
| Buyer-facing QR | Buyer scans to check in and complete disclosure flow |
| NFC side | Used for activation, setup, dashboard access, or event workflow |

The QR must point to a stable Rel8tion resolver URL, not a one-time event URL.

Preferred long-term endpoint:

```text
https://app.rel8tion.me/pass?code=PUBLIC_CODE
```

Until `/pass` is confirmed live, do not batch print more Event Pass QR codes.

## Standard Open House Workflow

### Step 1: Arrive Early

The LO should arrive 10-15 minutes before the open house begins.

Goal:

- Meet the agent calmly.
- Explain that this is a simple support tool.
- Set up the Event Pass before buyers arrive.

### Step 2: Introduce The Event Pass

Use this script:

> I brought a complimentary Rel8tion Event Pass for today's open house. It lets buyers check in by QR, complete the required disclosure flow, and sends you their info live by text so you are not chasing a paper sign-in sheet. If a buyer requests mortgage help, I can support them as the licensed loan officer available for the event.

### Step 3: Activate / Set Up The Event

The LO or agent should:

1. Open the Rel8tion setup flow.
2. Scan/tap the Event Pass as required.
3. Confirm the agent profile or enter the agent information.
4. Confirm the open house property.
5. Confirm the LO profile/availability if required.
6. Verify that the buyer QR opens the live event check-in page.
7. Run one quick test if appropriate.

### Step 4: Place The Event Pass

Place the Event Pass where buyers naturally enter or sign in.

Recommended agent instruction:

> When buyers walk in, just say: Please scan this first so I have your check-in and disclosure record.

### Step 5: Let The System Prove Itself

As buyers check in:

- Agent receives live SMS alerts.
- Buyer disclosures are completed digitally.
- Financing-help requests are routed only when buyer asks.
- LO is available to help without pressuring every buyer.

The LO should not hover or over-sell. The technology should create the opening.

### Step 6: End The Event And Show The Recap

At the end of the open house, show the agent the recap:

- Check-ins captured
- Disclosures completed
- Financing-help requests
- Represented buyers
- Follow-up opportunities
- Confirmation that live SMS alerts were sent

Then use the conversion script below.

## Agent Conversation Scripts

### Before The Open House

> I can stop by your open house and support buyers with mortgage questions if they need it. I will also bring a Rel8tion Event Pass so buyers can check in digitally, complete disclosures, and you get the buyer info live by text. It is quick and does not require you to download anything.

### When Arriving

> I brought the Event Pass we spoke about. It is very simple. Buyers scan the QR, check in, complete the disclosure flow, and you receive the buyer info live by text. If someone requests financing help, I am here to help them.

### If The Agent Is Skeptical

> No problem. You still get the buyer info live by text. This does not take over your open house. It just replaces the paper sign-in sheet and keeps the event organized afterward.

### If The Agent Says They Already Have A Sign-In Sheet

> Perfect. This is just a better version. Instead of trying to read handwriting later, the buyer info comes to your phone live, and the recap is organized after the event.

### If The Agent Worries About Leads Being Held Captive

> They are not held captive. You get the buyer info live by text during the open house. The recap and dashboard simply organize everything afterward.

### If The Agent Asks What It Costs

> Today is sponsored through my Rel8tion Event Pass. If you like it and want to use it without needing me to come out every time, you can activate your own dashboard and order the full Open House Kit.

### End Of Open House Conversion Script

> You had [X] check-ins today, [Y] disclosure records completed, and [Z] buyers who may need financing help. You already received the buyer info live by text. The recap email puts it all in one place. If you want to use this at your next open house without waiting for me to come out, you can activate your Rel8tion dashboard and order your Open House Kit now.

### Leaving The Event Pass With The Agent

> Keep this on your keys. At your next open house, call or text me and I can help you activate another sponsored Event Pass, or you can upgrade to your own permanent kit and run it yourself.

## Loan Officer Behavior Rules

### Do

- Be helpful and professional.
- Focus on supporting the agent's open house.
- Let buyers opt in for financing help.
- Use compliant language.
- Make it clear buyers can use any lender.
- Make it clear Rel8tion does not issue pre-approvals.
- Help the agent understand the recap and next step.
- Follow up after the event.

### Do Not

- Say you are buying leads.
- Say Rel8tion sells referrals.
- Promise guaranteed buyers, applications, or closings.
- Pressure buyers into mortgage conversations.
- Tell buyers they must use you.
- Collect SSNs, credit reports, bank statements, income documents, or loan applications through Rel8tion.
- Present Rel8tion as a lender or pre-approval provider.
- Make compensation sound tied to closed loans.
